Connect Google Drive

One-time setup to enable PDF uploads to your Drive.

1
Go to Google Cloud Console → APIs & Services → Credentials
2
Click Create Credentials → OAuth 2.0 Client ID → choose Web application
3
Under Authorized redirect URIs, add exactly:
http://localhost:5000/api/drive/callback
4
Copy the Client ID and Client Secret into your .env file as GOOGLE_OAUTH_CLIENT_ID and GOOGLE_OAUTH_CLIENT_SECRET
5
Restart the server (python app.py), then click Connect below.
